# NERDLE

## **Technology Stack**

- Typescript
- React
- Recoil
- MUI
- mathjs

## **Get Started**

### 1. Clone github repository

```sh
git clone https://github.com/nandemo3/nerdle.git
```

### 2. Install dependencies

```sh
# npm
cd nerdle
npm install
```

```sh
# yarn
cd nerdle
yarn install
```

### 3. Start Application

```sh
# npm
cd nerdle
npm start
```

```sh
# yarn
cd nerdle
yarn start
```
